## Anti-Patterns Verdict

Glassmorphism as default: confirmed - backdrop-blur-2xl on TaskItem, TasksHero, DropdownMenuContent.
Identical card grids: confirmed - FeatureGridHero renders icon+title+subtitle identical cards.
Hero-metric template: confirmed - TasksHero has big title, description, stats panel.
Floating particles in sidebar: 8 animate-float dots always visible.
Top-border gradient via-cyan-400 repeated on every card as template pattern.

## Priority Issues

[P0] Two incompatible design languages - PageHeroCard uses tokens, TasksHero/FeatureGridHero/TaskItem use hardcoded slate-900/80, white/10, cyan-500/20. Fix: migrate to CSS vars.

[P1] Glassmorphism as default - backdrop-blur-2xl on every Card violates DESIGN.md "Glass Is Exceptional" rule. GPU load on long task lists.

[P1] Floating particles and glow in sidebar - 8 animate-float particles, grid overlay, blur-sm glow on every active section. Cognitive fatigue for daily operators.

[P2] Destructive actions without confirmation - deleteTask() called directly, no AlertDialog or undo toast.

[P2] collapsedLabel="Недвижимость" for Parsing section - copy-paste error, shows wrong label in collapsed sidebar tooltip.

## Persona Red Flags

Power User: No error detail in TaskItem card. No keyboard shortcuts for retry. No bulk-retry.
First-Timer: Collapsed sidebar icon-only, collapsedLabel shows "Недвижимость". Particles distract from nav.
Developer: No task status filter. No pagination/count indicator.
