## Rules

### Enforce Three-Tier Layer Cleanliness
**Source**: 2026-06-17-09.50.md, 2026-06-17-14.00.md
**Rule**: During code reviews, reject pull requests where router files instantiate services/repositories or define service factory functions. Verify that factories reside in `dependencies.py`.

### Review Gateway Service Boilerplate
**Source**: 2026-06-17-14.00.md, 2026-06-17-14.30.md
**Rule**: Flag and reject custom forward/claims boilerplate code in Gateway Services. Verify that they inherit from `BaseGatewayService` and reuse standard methods.

### Detect Proxy Overengineering
**Source**: 2026-06-17-18.09.md, 2026-06-17-18.20.md
**Rule**: Ensure that classes do not act as dynamic proxy wrappers (lambdas). Check that domain services depend directly on specialized protocols/repositories instead of all-in-one proxy repositories.

### Debug Marker Cleanup Check
**Source**: 2026-06-17-09.50.md, 2026-06-17-18.09.md
**Rule**: Flag as ERROR any `logger.debug/info/warning` call containing a `[FIX]` prefix in modified files. These are temporary diagnostic markers and must be removed or rewritten as proper production observability before merge. Example finding: "`listings/dependencies.py:13 — logger.debug('[FIX] Instantiating...') must be cleaned up`".

### Router-Service Duplication Audit
**Source**: 2026-06-17-14.00.md
**Rule**: During review, scan router handler bodies for inline business logic exceeding 5 lines of data transformation or computation. Flag as WARN any such logic that belongs in the Service layer. Check especially for router methods that duplicate existing service methods.

### Post-BaseGatewayService Audit
**Source**: 2026-06-17-14.30.md
**Rule**: When a PR modifies `BaseGatewayService` (adds or changes methods), verify that all existing `*GatewayService` subclasses are reviewed for now-redundant method overrides. Flag as WARN any subclass that duplicates a method now available from the base class.

### File Size Limit Enforcement
**Source**: 2026-06-17-18.09.md
**Rule**: Flag as ERROR any non-config, non-migration, non-autogenerated Python file exceeding 150 lines. Include exact line count in the finding: e.g. `services/listings-service/app/modules/listings/import_service.py — 262 lines (limit: 150)`. Suggest decomposition strategy (extract validator, builder, or mapper modules).
