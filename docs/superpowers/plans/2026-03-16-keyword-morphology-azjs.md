# Keyword Morphology Az.js Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Добавить production-контур морфологии keywords через `Az.js`, чтобы `CommentKeywordMatch` создавался по словоформам ключевого слова без ручного дублирования падежей в БД.

**Architecture:** Основное слово остаётся в `Keyword`, а словоформы выносятся в отдельную таблицу `KeywordForm` с источником `generated|manual` и отдельным механизмом исключений для удалённых autogenerated-форм. Backend через `Az.js` генерирует формы, сохраняет их, использует в comments matching pipeline и предоставляет операции пересчёта/редактирования.

**Tech Stack:** NestJS, Prisma, PostgreSQL, Az.js, React, Vitest
