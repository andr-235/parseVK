# Product

## Users

Сотрудники, занимающиеся выявлением противоправных высказываний в интернете. Операторы-аналитики, которые ежедневно мониторят соцсети (ВК, ТГ, ОК), работают с таблицами, фильтрами, очередями задач, экспортом. Основной сценарий — видеть что происходит прямо сейчас, управлять задачами парсинга, быстро находить аномалии и нарушения в комментариях и постах.

## Product Purpose

Платформа социальной аналитики для поиска, мониторинга и выявления противоправных высказываний в комментариях и постах. Парсит, хранит, анализирует и визуализирует данные из ВКонтакте, Telegram и OK. Позволяет управлять задачами парсинга, системой ключевых слов с морфологией, отслеживать авторов, модерировать контент, экспортировать данные.

## Brand Personality

Деловой, креативный, современный. Уверенный и серьёзный инструмент, без излишней строгости — с характером и визуальной культурой.

## Anti-references

- Клоунские цвета (кислотные, неестественные, игрушечные палитры)
- Лендинг-оформление: hero-секции, большие декоративные заголовки, маркетинговые блоки
- MVP-стиль: необработанный, недоработанный, «как есть» UI
- Любая стилизация «под консоль/терминал»
- Игровые/развлекательные визуальные приёмы

## Design Principles

1. **Data First** — контент и данные определяют композицию. Каждый элемент интерфейса существует ради данных или действия с ними.
2. **Semantic Color** — цвет используется только для передачи состояния или действия. Никакой декоративной раскраски.
3. **One Command Accent** — один акцентный цвет на экран. Второй цвет только для смыслового контраста (статусы, семантика).
4. **Border Before Shadow** — сначала граница для иерархии, тень только когда нужна глубина.
5. **Инструмент, не витрина** — интерфейс рабочего места, а не маркетинговая презентация. Никаких декоративных карточек, градиентного текста и glassmorphism.
